# Brand Guidelines: DevOps Buddy

- **Theme:** Dark Mode First (Vercel/Supabase aesthetic).
- **Primary Color:** Electric Blue (#0070F3) for primary actions.
- **Background:** Deep Space Black (#0A0A0A) with subtle gray borders (#1A1A1A).
- **Typography:** Inter or Geist Sans. Clean, sans-serif, highly legible.
- **UI Elements:** Use glowing borders for active "Self-Healing" states, clean tables for the Q-Value/RL State memory, and a split-pane view (Logs on the left, AI Reasoning/PR generation on the right).
